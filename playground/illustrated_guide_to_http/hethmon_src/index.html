Illustrated Guide to HTTP

Source Code

The source code for the book is organized by the HTTP level of the server and by the operating system. Included here are HTTP servers for 0.9, 1.0 and 1.1 levels. Both servers are available for OS/2 and NT. Also included are CGI tester programs and a simple HTTP/1.1 client test program.

  • CGI Test Program for OS/2
  • CGI Test Program for NT
  • Digest Access Authentication Code
  • HTTP/0.9 Server for OS/2
  • HTTP/0.9 Server for NT
  • HTTP/1.0 Server for OS/2
  • HTTP/1.0 Server for NT
  • HTTP/1.1 Server for OS/2
  • HTTP/1.1 Server for NT
  • HTTP/1.1 Client Tester for OS/2
  • HTTP/1.1 Client Tester for NT
